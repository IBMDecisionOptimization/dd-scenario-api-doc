Welcome to the dd-scenario-client-python documentation!
=======================================================

This Python client makes it easy to work with the Decision Optimization Scenario
REST API.

The most important class is :class:`~dd_scenario.Client`. First create a
:class:`~dd_scenario.Client` then use its methods to interact with the scenario
service.
